# Wild Suits

**Wild Suits** is a game developed for **Godot Wild Jam 89**.  

You can play it on [itch.io](https://aaatum-games.itch.io/wild-suits).

---

## Description

**A new game to play at the casino?**

Each round, the cards gain a new tactical purpose by repurposing enemy abilities.  
This game was made in **nine days** for **Godot Wild Jam #89** with the theme **Repurposed**.

### Objective

- Defeat all enemies
- Win **5 rounds** to complete the game

### Cards

- You use cards to take actions
- Cards can only be played on **matching tile types**

### Abilities

Each card allows you to perform a different action:

- **Move** – Move 1 tile
- **Swap** – Switch places with any enemy on the board
- **Slash** – Kill an adjacent enemy
- **Shoot** – Kill an enemy in the same row or column
- **Push** – Push an enemy to the edge of the board
- **Shield** – Become immune to enemy actions for 1 turn
- **Trap** – Place a trap that kills any unit that steps on it

### Extras

- If you run out of cards, you lose
- The character you choose at the beginning makes their ability cards more common to draw
